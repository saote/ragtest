Quick stop（急停)
功能简介
Quick stop 即急停，当驱动器处于 Operation Enable 状态下，在触发报错或限位等情况时，将根据设定的 Quick stop 模式进行急停动作，动作完成后将处于设定的 Quick stop 模式下状态，用户可根据应用场景适配 Quick stop 模式。

快速停止动作模式 605Ah-00h
快速停机处理：
| 值 | 描述                                                                                                               |
| ------ | ---------------------------------------------------------------------------------------------------------------------- |
| 0      | 自由停机，保持自由运行状态                                                                                             |
| 1      | 以 0x6084-0x00（轮廓减速度）设定的减速度斜坡停机（回零模式下则为 0x609A-0x00，回零减速度），停机完成后保持自由运行状态 |
| 2      | 以 0x6085-0x00（快速停止减速度）设定的减速度斜坡停机。停机完成后保持 switch on disabled                                |
| 3      | 以 60C6h-00h（最大减速度）紧急停止转矩停机，停机完成后保持自由运行状态                                                 |
| 5      | 以 0x6084-0x00（轮廓减速度）设定的减速度斜坡停机（回零模式下则为 0x609A-0x00，回零减速度），完成后保持位置锁定状态     |
| 6      | 以 0x6085-0x00（快速停止减速度）设定的减速度斜坡停机，停机完成后保持位置锁定状态                                       |
| 7      | 以 60C6h-00h（最大减速度）紧急停止转矩停机，停机完成后保持位置锁定状态                                                 |
快速停止减速度 6085h-00h
快速停止减速度在用 quick stop 方式 2、6 停机时使用，电机按照快速停止减速度减速到 0。
功能应用
Quick stop 指令可以通过字典指令触发也可以通过 IO 触发。
指令触发：0x6040-0x00-bit2 为 0。
IO 触发：可通过设置 IO 输入触发事件为急停，通过触发 IO 来触发急停。
Quick stop active 状态时 0x6041-0x00-bit5 为 0。
